# Network Scanner

## Description

A Python-based Network Scanner designed to identify active hosts within a specified network range. The tool helps administrators discover devices connected to a network.

## Features

* Host discovery
* Network scanning
* Active device detection
* Quick network analysis
* Command-line interface

## Technologies Used

* Python 3
* Socket Programming
* Kali Linux

## How to Run

```bash
python3 network_scanner.py
```

## Learning Outcomes

* Network discovery
* Host identification
* Network analysis
* Cybersecurity fundamentals
